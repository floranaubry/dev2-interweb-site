# Interweb

Nuxt 4 + TailwindCSS + i18n + Page Builder.

## Requirements

- **Node.js** ≥ 20 (LTS recommended)
- **pnpm** ≥ 9 (via Corepack)

## Quick Start

```bash
corepack enable        # Enable pnpm (once per machine)
pnpm install           # Install dependencies
pnpm dev               # Development server → http://localhost:3000
```

## CI / One Command

```bash
pnpm check             # Format + Types + Guards + Build (single entrypoint)
```
