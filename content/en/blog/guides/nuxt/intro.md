# Introduction to Nuxt 4

This guide walks you through your first steps with Nuxt 4.

## Prerequisites

- Node.js 18+
- pnpm (recommended)
- Basic Vue.js knowledge

## Installation

```bash
pnpm dlx nuxi@latest init my-project
cd my-project
pnpm install
pnpm dev
```

## Project Structure

Nuxt 4 uses an `app/` structure:

- `app/pages/` — Automatic routes
- `app/components/` — Vue components
- `app/layouts/` — Page layouts
- `app/composables/` — Reusable logic

## Conclusion

Nuxt 4 simplifies modern Vue.js application development with minimal configuration.
