# Introduction à Nuxt 4

Ce guide vous accompagne dans vos premiers pas avec Nuxt 4.

## Prérequis

- Node.js 18+
- pnpm (recommandé)
- Connaissances de base en Vue.js

## Installation

```bash
pnpm dlx nuxi@latest init mon-projet
cd mon-projet
pnpm install
pnpm dev
```

## Structure du projet

Nuxt 4 utilise une structure `app/` :

- `app/pages/` — Routes automatiques
- `app/components/` — Composants Vue
- `app/layouts/` — Layouts de pages
- `app/composables/` — Logique réutilisable

## Conclusion

Nuxt 4 simplifie le développement d'applications Vue.js modernes avec une configuration minimale.
